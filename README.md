# Indoor Navigation for Visually Impaired

## Overview
An assistive indoor navigation system using BLE beacons and object detection
to help visually impaired users navigate indoor spaces.

## Hardware & Software
- ESP32
- BLE Beacons
- MobileNet Object Detection
- Android (MIT App Inventor)

## Features
- RSSI-based indoor localization
- Real-time obstacle detection
- Voice-guided navigation feedback

## System Flow
1. BLE beacons transmit signals
2. ESP32 calculates RSSI values
3. Mobile app estimates position
4. Object detection identifies obstacles
5. Voice feedback guides the user

## Achievement
Finalist at ACCS Challenge, IIIT Bengaluru.
